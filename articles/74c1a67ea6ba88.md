Cursor アップデートしたらエディタの右上付近にある Claude Code のコンソール開いたり Markdown のプレビュー画面開いたりできるアイコンが消えたので、それを表示させる方法の備忘録です。

![アイコン表示例の画像](https://storage.googleapis.com/zenn-user-upload/cdbe3f609c38-20251121.png)

# 表示方法

`三点リーダーアイコンをクリック` → `Configure Icon Visibilityをクリック` → `任意の項目を選択` でアイコンを表示できます。トグル方式なので同じ手順でチェック済みの項目を再選択するとアイコンが非表示になります。

![アイコン表示・非表示の操作gif](https://storage.googleapis.com/zenn-user-upload/f700b7dce26a-20251121.gif)

自分はショートカットとか使いこなせないマンなので、いきなりアイコン消えて絶望しましたが、これでまた快適に開発できそうです！
